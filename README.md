# UIT Mobile

A vibe coded flutter app for android (13+) and iOS (15+)

> No further development

## Installation

Go to [Release](../../releases/latest)

Trollstore quick link: [![Trollstore](https://img.shields.io/badge/trollstore-89b4fa?style=for-the-badge&logoColor=cdd6f4)](https://tinyurl.com/uit-mobile-ipa)

> [!NOTE]
>
> - The IPA is unsigned. You need to sign it before installing. Or you can install with Trollstore, Jailbreak, etc...
> - With trollstore link, you have to enable trollstore "URL Scheme" in Trollstore's security
